Веб приложение для отправки SMS.

Технологии
 .asp net core 5, ef core + postgres, RESTful

Реализовано простое SPA web-приложение без аутентификации для отправки смс сообщений.
Главный экран содержит 2 области:  
Область для отправки нового СМС сообщения с полями: 
•	Текст SMS сообщения 
•	Номер получателя сообщения
•	Имя отправителя
•	Кнопка отправки
Область с списком отправленных сообщений. Список содержит колонки:
•	Дата и время отправки
•	Номер получателя
•	Текст сообщения
•	Статус сообщения (Отправлено, Доставлено, Ошибка отправки)
web интерфейс приложения с использованием React на typescript.
хранение SMS сообщений в базе данных postgresql c использованием ef core.
отправку и получение статусов сообщений посредством реализации фейкового сервиса отправки (генерация рандомного статуса отправки).
