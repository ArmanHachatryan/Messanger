Веб приложение для отправки SMS.

Технологии
 .asp net core 5, ef core + postgres,

Требования
Реализовать простое SPA web-приложение без аутентификации для отправки смс сообщений.
Главный экран должен содержать 2 области:  
Область для отправки нового СМС сообщения с полями: 
•	Текст SMS сообщения (обязательное)
•	Номер получателя сообщения (обязательное)
•	Имя отправителя
•	Кнопка отправки
Область с списком отправленных сообщений. Список должен содержать колонки:
•	Дата и время отправки
•	Номер получателя
•	Текст сообщения
•	Статус сообщения (Отправлено, Доставлено, Ошибка отправки)
Реализовать web интерфейс приложения с использованием React на typescript.
Реализовать хранение SMS сообщений в базе данных postgresql c использованием ef core.
Реализовать отправку и получение статусов сообщений посредством реализации фейкового сервиса отправки (например, генерация рандомного статуса отправки).
